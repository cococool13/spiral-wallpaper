Foundation

SPR-00 · Why the brand exists

Every rule in this document traces back to the founder's brief. When a decision isn't covered here, go back to these words and derive it.

"I've always liked spiral staircases because they are complex but also simple." — on the name
"Think like walking into a concrete warehouse with concrete floors, not a lot of walls, metal beams — super modern, sophisticated, and professional." — on the aesthetic
"I love optimizing PCs for gaming. My app is optimized that way as well. We don't sacrifice on the look of it, but everything that can be optimized for better performance, we do." — on engineering

Privacy

Everything is privacy-first. No account wherever possible. Nothing leaves the machine unless the user sends it.

Ease of use

Onboarding as simple as possible. Nothing is left "obvious" — if the app runs after the window closes, it says so. Everything is stated.

Super lightweight

Optimized like a gaming PC: never sacrifice the look, but everything that can be faster, is. This guide itself ships as one 45 KB file.

Free for everyone

Totally free for users, and as close to free to run as possible. The brand never sells; it states.

The Mark

SPR-01 · Identity

A ribbon folded into an S — one strand turning around itself, read equally as a spiral staircase in section and a DNA helix. Complex but simple, like the staircases the brand is named for. It is flat, geometric, and set in one color at a time: helix red by default, ink or paper where red can't live. Pure vector — the same file works from a billboard to a favicon.

A. Primary — red on paper / concrete
B. Reversed — paper on steel
C. Clearspace — ¼ mark height, all sides
D. Scale — one file, every size

Menu bar and favicon use the ink or template version at 16–24px. No separate glyph needed.

E. Never
  • Use any color beyond helix red, ink, or paper
  • Add gradients, shadows, bevels, or outlines
  • Rotate or mirror it — the ribbon folds one way
  • Place it on photography or busy texture
  • Set the mark and wordmark in different reds
F. Misuse — one wrong example teaches more than ten right ones

Lockup

SPR-02 · Mark + Wordmark
A. Vertical lockup — drawn letterforms
B. Rules
  • The lockup's letterforms are drawn — never retype "SPIRAL" in Archivo inside a lockup
  • Mark in helix red, letters in ink; reversed: everything paper
  • Use the lockup on the website header, packaging, and first-run screens
  • Inside apps, the mark stands alone — one brand, one website, many apps

Material

SPR-03 · Color

The palette is the warehouse with one object in it: poured concrete floors, mill steel beams, and helix red as the only color. Red is for the mark, interaction, and warnings. If a screen is more than a few percent red, something is wrong. One more material exists but has no swatch: glass. Controls — buttons, and only buttons — are frosted glass floating over the concrete; surfaces never are.

Type

SPR-04 · Typography

One family, many widths — complex but simple, the same move as the mark. Archivo's variable width axis goes wide and heavy for display, normal for reading. IBM Plex Mono handles anything that behaves like data: labels, tokens, file sizes, version numbers. Both are open-source and free, so the brand costs nothing to license. The lockup's drawn letterforms are reserved for the lockup itself.

Space

SPR-05 · Layout

An 8px base unit, doubled and halved — never eyeballed. Not a lot of walls: generous padding, hairline formwork borders, flat surfaces, few dividers. Depth comes from material contrast. Blur and shadow exist in exactly one place — glass, the control layer that floats above the concrete.

Motion

SPR-06 · Behavior
  • Motion explains state; it never decorates. If nothing changed, nothing moves.
  • One easing curve everywhere: cubic-bezier(.2,.7,.2,1), 150–400ms.
  • Entrances rise — the direction the ribbon climbs. Exits fade.
  • prefers-reduced-motion is honored in every app, always.
  • Optimized like a gaming PC: an animation that costs a frame is cut. Never sacrifice the look — but everything that can be faster, is.

In Use

SPR-07 · Components & applications

The brand at work. Buttons state exactly what happens. The download button auto-detects the platform, so the label already knows the answer. Windows say what they do — including what keeps running after they close.

A. Buttons — liquid glass over the material

Pill radius, frosted blur, one specular sheen. Primary: tinted helix glass, deepening to oxblood when pressed. Secondary: clear glass. The site auto-detects the platform and shows one primary button — its icon and label already know the answer. Never both platforms at once outside this spec sheet.

Voice

SPR-08 · Writing

The voice comes from one rule in the founder's brief: nothing that matters is "obvious" — it needs to be stated. Spiral states; it never sells.

Say
  • "No account needed."
  • "Closing this window keeps Spiral running. Quit fully from the menu bar."
  • "Deletes 1.2 GB of caches. Nothing else."
  • "Free. No catch."
Never say
  • "Supercharge your workflow" — sell nothing, state everything
  • "Oops! Something went wrong" — errors name the problem and the fix
  • "We value your privacy" — prove it in the sentence instead
  • Anything a beam wouldn't say

Tokens

SPR-09 · Drop-in CSS

Paste this into every Spiral app. If a value isn't here, it isn't in the brand.

:root {
  /* material */
  --conc-01: #EBE9E4;   /* page */
  --conc-02: #DDDAD3;   /* surface */
  --conc-03: #CFCCC4;   /* border */
  --ink-01:  #10181B;   /* ink — all body text */
  --stl-02:  #666863;   /* secondary text — 4.7:1 on page */
  --hlx-01:  #D52E2B;   /* helix red — mark, accents, never body copy */
  --hlx-02:  #6F1011;   /* oxblood — pressed / hover */

  /* type */
  --font-ui:   "Archivo", sans-serif;
  --font-mono: "IBM Plex Mono", monospace;
  --wdth-display: 125;  --wght-display: 850;
  --wdth-heading: 112;  --wght-heading: 700;

  /* rhythm */
  --unit: 8px;
  --ease: cubic-bezier(.2, .7, .2, 1);
  --dur-fast: 150ms;  --dur-slow: 400ms;

  /* glass — controls only, never surfaces */
  --radius-ctl: 999px;
  --glass-blur: blur(16px) saturate(1.7);
  --glass-edge: rgba(255, 255, 255, .45);
  --glass-sheen: linear-gradient(115deg,
      rgba(255,255,255,.55), rgba(255,255,255,0) 55%);
}
